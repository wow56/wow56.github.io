# Docker Compose

# 简介

  • Compose 是用于定义和运行多容器 Docker 应用程序的工具。
  • 通过 Compose，可以使用 YML 文件来配置应用程序需要的所有服务。
  • 然后，使用一个命令，就可以从 YML 文件配置中创建并启动所有服务。

# 使用步骤

  1. 使用 Docker 定义应用程序的环境。
  2. 使用 docker-compose.yml 定义构成应用程序的服务，这样它们可以在隔离环境中一起运行。
  3. 最后，执行 docker-compose up 这个命令来启动并运行整个应用程序。

  • docker-compose.yml 配置案例如下：

    # yaml 配置实例
version: '3'
services:
  web:
    build: .
    ports:
    - "5000:5000"
    volumes:
    - .:/code
    - logvolume01:/var/log
    links:
    - redis
redis:
  image: redis
volumes:
logvolume01: {}
