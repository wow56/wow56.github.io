# 自动提示添加 serialVersionUID

  1. Preferences -> Inspections -> Serialization issues -> Serialization class without 'serialVersionUID' 打上勾
  2. 光标放在实现了 Serializable 接口的类上，然后 Alt + Enter，就会出现提示
