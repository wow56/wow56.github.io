# 语言/语法

语言/语法模块

开发语言主要分为两类：

  • 解释型语言
    • 需要解释器，每次执行的时候，都需要解释一句，执行一句，效率低。
  • 编译型语言
    • 将程序直接编译成机器语言，每次执行都是直接运行编译后的文件，效率高。
